Web Design

案件流程
  1. 需求了解

    溝通討論、報價單開立
    在最一開始的階段，專案企劃會帶領您一步步起始構思，提供案例與分享製作方向的建議，底定網頁的方向與設計風格。

  2. 正式簽約

    確認服務內容，報價單、合約書簽署
    在最一開始的階段，愛貝斯的專案企劃會帶領您一步步起始構思，提供案例與分享製作方向的建議，底定網頁的方向與設計風格。

  3. 設計討論 注意事項

    訪談想法與素材提供 透過訪談與線框圖輔助討論，確認設計風格細節、功能、瀏覽動線。並由客戶提供設計素材以利於後續設計版型

  4. 視覺設計 注意事項

    網站版型 UI/UX 設計與校對
    緊扣企業品牌精神及產業需求，視覺設計師以專業且洞悉設計趨勢的美學概念提供視覺強化的解決方案，打造符合您的形象網站。

  5. 前端效果 注意事項

    前端切版、架構建置與動態效果製作並導入RWD響應式技術
    前端工程師掌握使用者操作需求及網站互動研究，專注前端切版、把關網站瀏覽流暢性，為一頁頁靜態版面注入生命。

  6. 程式開發

    客製化功能程式建置
    後端工程師致力更人性化的網站及管理系統的研發，讓網站擁有堅固穩定的系統基底，並且製作好用的後台讓客戶可自行更換資料。

  7. 正式上線與保固

    網站成品校對與上架並正式結案上線
    專案企劃協助掌控各階段執行狀況與網站成品把關，協助您完成網站建置的一道道關卡，讓網站正式上線!

  8. SEO策略與經營

    上線後是另一個開始
    透過愛貝斯網頁設計簡單易用的後台，您已經擁有優秀的SEO體質，但別忘了為您的網站每頁撰寫專屬Meta TDK，強化SEO效果。

注意事項
  • 設計討論

    依合作相關係，線稿風格等由業務與客戶討論與確認。
    若在設計畫面前發現有不合理的地方皆可提出討論修改。
    建議先了解清楚後再執行"視覺設計"

    P.S. 線稿皆為參考不是絕對

  • 視覺設計
    • 一般開版尺寸為 寬: 1920 x 高: 1080 (最小高度)
    • 檔案結構範例 
      project // 專案資料夾
    ├─ psd (ai, XD) // 設計檔
    └─ data // 相關資料
        ├─ img // 圖片相關
        │   ├─ pic // logo ... 等客戶提供圖片
        │   └─ is // istockphoto 圖庫
        ├─ fonts // 使用的字型
        └─ word // 文案相關文件
    • 可用 AI, PS, XD 等軟體製作。 檔案內的圖片不要合併，文字也不要做點陣化或轉曲。 區塊請善用圖層管理 (如: menu， footer ...)
  • 前端效果

    • 有喜歡的或想做的特效可與前端工程師討論 建議可直接提供參考連結進行討論
