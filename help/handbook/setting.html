Setting Variable

匯入輔助變數 

@import helpers/variable/_var-setting
Media 斷點

可自行設定斷點值 

$xl: 1200
$lg: 992
$md: 768
$sm: 575

$rwd: 1200 //Menu轉換手機尺寸
$rwd-pc: 1366 //電腦尺寸
$rwd-pcs: $xl //小電腦尺寸
$rwd-pad: $lg //平板尺寸
$rwd-pads: $md //小平板尺寸
$rwd-phone: $sm //手機尺寸
$rwd-phones: 320 //小手機尺寸

參考尺寸

裝置 尺寸
PC 1920x1080, 1680x1050, 1600x900, 1440x900, 1366x800, 1366x768, 1024x768
Pad 800x1280, 768x1024, 600x1024, 600x962
Phone [iPhone 6/7/8 Plus, 414x736] [iPhone 6/7/8, 375x667] [iPhone 5/SE, 320x568]
Default

module default, 預設字級 16px 行高 24px 

$default: (htmlFz: 10, bodyFz: 16, bodyLh: 24, bodyCr: #333)
Font

Attributes: font( name )

文字字體、icon 等相關設定。

$font-main 主要字體設定
$font 外嵌字體設定，使用方式參考helpers font( name ) 
// Font Main  預設 ( Arial, 'Microsoft JhengHei', sans-serif )
$font-main: Arial, 'Microsoft JhengHei', sans-serif !default

// import font
$font: ()

Font Size (u-fz)

Class: u-fz-[val]

設定文字大小

SASS 
$fzArray: 30, 20, 18, 16, 14, 12
CSS 
.u-fz-30 { font-size: 30px; }
.u-fz-20 { font-size: 20px; }
.u-fz-18 { font-size: 18px; }
.u-fz-16 { font-size: 16px; }
.u-fz-14 { font-size: 14px; }
.u-fz-12 { font-size: 12px; }
Color

Attributes: cr( name ) , crc( name )

常用顏色相關設定。(參照boostorp 色碼)。
使用方式參考helpers cr( name ) , crc( name )

$color: () 基礎共用顏色
$colorCustom() 自訂CIS顏色
Wrap(l-wp)

Class: .l-wp-[val] || .l-wp-[name]

module layout, Wrap 相關設定。
主要控制內容寬度 

$wpRange: 1024 // 基本範圍
$wpBoth: 15 // 左右間距(px)
$wpCustomName: false
$wpArray: 1680, 1440, 1366, 1024
設定方式

有兩個模式，1.自行命名。2.預設名稱。
因主要採BEM的方式命名，產出的class皆為".l-wp-[name]"前方的綴詞是固定的

變數 值 說明
$wpCustomName false, true 是否自行命名開關
$wpArray 0 ~ 99+ 陣列(輸入數值即可)
$wpCustomName: false

Class: .l-wp-[val]

SASS 
$wpCustomName: false
$wpArray: 1680, 1440
CSS 
.l-wp-1680 { max-width: 1680px; }
.l-wp-1440 { max-width: 1440px; }
$wpCustomName: true

Class: .l-wp-[name]

SASS 
$wpCustomName: true
$wpArray: (max1680:1680, max1440:1440)
CSS 
.l-wp-max1680 { max-width: 1680px; }
.l-wp-max1440 { max-width: 1440px; }
Grid(l-gd)

Class: .l-gd-row[val]

網格相關設定 (p.s. 已有自適應)

SASS 
$gridAarry: 2, 3, 4
CSS 
.l-gd-row2 > * { width: calc(100%/2); }

.l-gd-row3 > * { width: calc(100%/3); }

.l-gd-row4 > * { width: calc(100%/4); }
Display(u-display)

Class: .u-display-[val], .u-display-[size]-[val]

display 顯示 & 隱藏設定

$displaySwitch 各載具自適應開關
t: 開啟, f: 關閉		 sm, md, lg, xl尺寸設定在Media 
$displaySwitch: (sm: f, md: f, lg: f, xl: f)
Margin(u-m)

Class: .u-m-[val], .u-m-[size]-[val]

外距輔助工具設定

$mgUnit 單位設定
預設: rem		 px, em, rem, vw, %
$mgSwitch 各載具自適應開關
t: 開啟, f: 關閉		 base預設為開啟, sm, md, lg, xl尺寸設定在Media
$mgArray 設定方向&數值
  • l: left
  • r: right
  • t: top
  • b: bottom
  • x: left & right
  • y: top & bottom
 
$mgUnit: rem
$mgSwitch: (base: t, sm: f, md: f, lg: f, xl: f)
$mgArray: (l: 10, r: 10, t: 10, b: 10, x: 10, y: 10)
Padding(u-p)

Class: .u-p-[val], .u-p-[size]-[val]

內距輔助工具設定

$pdUnit 單位設定
預設: rem		 px, em, rem, vw, %
$pdSwitch 各載具自適應開關
t: 開啟, f: 關閉		 base預設為開啟, sm, md, lg, xl尺寸設定在Media
$pdArray 設定方向&數值
  • l: left
  • r: right
  • t: top
  • b: bottom
  • x: left & right
  • y: top & bottom
 
$pdUnit: rem
$pdSwitch: (base: t, sm: f, md: f, lg: f, xl: f)
$pdArray: (l: 10, r: 10, t: 10, b: 10, x: 10, y: 10)
