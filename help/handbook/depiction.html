主要命名規則

採BEM + 空間命名

B__E--M
為雙底連接塊和塊的子元素。
雙中線依舊用來描述狀態。

U單元 unit-B區塊 block_E元素 element--M修飾 Modifier

大致如下：

  1. B__E--M
  2. U-B__E--M
  3. U-B-depiction__E--M
  4. U-B__E-depiction--M
  5. U-B__module
  6. U-B__E module

空間命名

這裡只選用 l- m- u- is/has- js-

  • l- Layout 佈局

  • 3擇1

    • o- Objects 物件

    • e- Element 元素

    • u- Unit 零件

  • 2擇1 [ 選用 m- ]

    • c- Component 組件

    • m- Module 模塊

  • 3擇1 [ 選用 u- ]

    • u- Utility 實用工具

    • h- Helpers 幫手

    • f- Fuctions 功能

  • is/has- States 狀態

  • js- JavaScript hooks

Layout 佈局

綴詞加上 l-。# 沒有綴詞視為獨立 block or element

  1. wrap 包裹 l-wp
  2. content 內容 or container 容器 l-ctr
  3. section 部分 l-stn
  4. block 塊 l-bk
  5. box 框 l-bx
  6. inner 內 l-ir

wrap > content > section > block > box > inner

Module 模塊

Jhuangpin helpers 所編窵的組件。 綴詞 m-

  • banner
  • card
  • button
  • ...
Utility 實用工具

獨立常用樣式, 輔助工具。 綴詞 u-

  • Margin/padding
  • Text (colors, size, styles)
  • Common Background Colors
  • Hiding/Showing stuff
  • Display (block, inline-block)
  • ...
