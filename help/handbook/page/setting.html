設定 Setting

步驟
  1. 下載 JhuangPing Heplers
  2. 將config資料夾放入sass(scss)內
// Config
@import config/_config

// layout
@import layout/_module
@import layout/_header
@import layout/_footer

// page
@import layout/page/_index

結構 Skeleton

config/
├─ helpers/
│  ├─ function/
│  ├─ mixin/
│  └─ variable/
└─ module/

  • SASS

    • config

      JhuangPing 設定

      • helpers

        助手

        • function

          功能

        • mixin

          函式

        • variable

          變數
        • _helpers-setting.sass助手設定檔

      • module

        模組
        • _module-setting.sass 模組設定檔
        • _md-reset.sass 初始化
        • _md-default.sassHTML 預設值
        • _md-font.sass文字,icon設定
        • _md-common.sass共用
        • _md-javascript.sassjquery 生成class
        • _md-layout.sass 佈局
        • _md-keyframes.sass 影格
        • _md-typography.sass文字標籤
        • _md-button.sass按鈕
        • _md-grid.sass網格
        • _md-banner.sass廣告
        • _md-path.sass麵包屑
        • _md-page.sass分頁
        • _md-form.sass表單
        • _md-table.sass表格樣式
      • _config.sass 設定

    • layout

      排版

      • page

        內頁
      • _module.sass 共用模組
      • _footer.sass 頁腳
      • _header.sass 選單
