config

  • Config

    匯入輔助變數 

    @import helpers/variable/_var-setting

  • Media 斷點

    可自行設定斷點值 

    $rwd: 1200 //Menu轉換手機尺寸
$rwd-pc: 1280 //電腦尺寸
$rwd-pcs: 1024 //小電腦尺寸
$rwd-pad: 992 //平板尺寸
$rwd-pads: 768 //小平板尺寸
$rwd-phone: 480 //手機尺寸
$rwd-phones: 320 //小手機尺寸

    參考尺寸

    裝置 尺寸
    PC 1920x1080, 1680x1050, 1600x900, 1440x900, 1366x800, 1366x768, 1024x768
    Pad 800x1280, 768x1024, 600x1024, 600x962
    Phone [iPhone 6/7/8 Plus, 414x736] [iPhone 6/7/8, 375x667] [iPhone 5/SE, 320x568]

  • Font

    文字字體、icon 等相關設定。

    $font-main 主要字體設定
    $font 外嵌字體設定，使用方式參考helpers font() 
    // Font Main  預設 ( Arial, 'Microsoft JhengHei', sans-serif )
$font-main: Arial, 'Microsoft JhengHei', sans-serif !default

// import font
$font: ()

  • Color

    常用顏色相關設定。(參照boostorp 色碼)。
    使用方式參考helpers cr() , crc()

    $color: () 基礎共用顏色
    $colorCustom() 自訂CIS顏色

  • Default

    module default, 預設字級 16px & 行高 24px 

    $default: (fz: 16, lh: 24)

  • Container(ctr)

    module layout, Container 相關設定。
    主要控制內容寬度 

    $ctrRange: 1024 // 基本範圍
$wpBoth: 15 // 左右間距(px)

    設定方式

    有兩個模式，1.自行命名。2.預設名稱。
    因主要採BEM的方式命名，產出的class皆為".l-wp-[name]"前方的綴詞是固定的

    變數 值 說明
    $wpCustomName false, true 是否自行命名開關
    $wpArry 0 ~ 99+ 陣列(輸入數值即可)
    $wpCustomName: false

    class: .l-wp-[val]

    SASS 
    $wpCustomName: false
$wpArry: 1680, 1440
    CSS 
    .l-wp-1680 { max-width: 1710px; }
.l-wp-1440 { max-width: 1470px; }
    $wpCustomName: true

    class: .l-wp-[name][val]

    SASS 
    $wpCustomName: true
$wpArry: (max1680:1680, max1440:1440)
    CSS 
    .l-wp-max1680 { max-width: 1710px; }
.l-wp-max1440 { max-width: 1470px; }

  • Grid(c-gd)

    網格相關設定 (p.s. 已有自適應)

    SASS 
    $gridArry: 2, 3, 4
    CSS 
    .c-gd-row2 > * { width: calc(100%/2); }

.c-gd-row3 > * { width: calc(100%/3); }

.c-gd-row4 > * { width: calc(100%/4); }

    @import helpers/_helpers-setting
