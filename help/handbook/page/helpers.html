media
段點 @media

針對指定的「媒體」設定樣式，這裡使用的媒體類型為" all " ( 用於所有設備 )。

裡頭的$rwd 等變數可在_config.sass修改設定。

預設值：

變數 $rwd $rwd-pc $rwd-pcs $rwd-pad $rwd-pads $rwd-phone $rwd-phones
預設值 1024 1280 1024 992 768 480 320

在這裡有三個可用分別是

  1. @media all and (max-width) 
    +rwdw-max(val)

    使用方式：

    SASS 
    .test
    width: 50%
    +rwdw-max()
        width: 100%
    CSS 
    .test {
    width: 50%;
}
@media all and (max-width: 1024px) {
    .test {
        width: 100%;
    }
}
  2. @media all and (min-width) 
    +rwdw-min(val)

    使用方式：

    SASS 
    .test
    width: 50%
    +rwdw-min()
        width: 100%
    CSS 
    .test {
    width: 50%;
}
@media all and (min-width: 1025px) {
    .test {
        width: 100%;
    }
}
  3. @media all and (min-width) and (max-width) 
    +rwdw-range(max,min)

    使用方式： 

    +rwdw-range(1024,768)
