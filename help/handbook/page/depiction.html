主要命名規則

採BEM + 空間命名OOCSS

原生 B__E--M 為雙底連接塊和塊的子元素，再此簡化為單底連接。
雙中線依舊用來描述狀態

U單元 unit-B區塊 block_E元素 element--M修飾 Modifier

大致如下：

  1. B_E--M
  2. U-B_E--M
  3. U-B-depiction_E--M
  4. U-B_component

空間命名OOCSS

這裡只選用 l- e- c- u- is/has- js-

  • l- Layout 佈局

  • 3擇1 [ 選用 e- ]

    • o- Objects 物件

    • e- Element 元素

    • u- Unit 零件

  • 2擇1 [ 選用 c- ]

    • c- Component 組件

    • m- Module 模塊

  • 3擇1 [ 選用 u- ]

    • u- Utility 實用工具

    • h- Helpers 幫手

    • f- Fuctions 功能

  • is/has- States 狀態

  • js- JavaScript hooks

Layout

綴詞加上 l-。# 沒有綴詞視為獨立 block or element

  1. wrap 包裹 l-wp
  2. content 內容 or container 容器 l-ctr
  3. section 部分 l-stn
  4. block 塊 l-bk
  5. box 框 l-bx
  6. inner 內 l-ir

wrap > content > section > block > box > inner
