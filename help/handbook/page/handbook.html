media
段點 @media

針對指定的「媒體」設定樣式，這裡使用的媒體類型為" all " ( 用於所有設備 )。

在這裡有三個可用分別是

  1. @media all and (max-width) 
    +rwdw-max(val)

    使用方式：

    SASS 
    .test
    width: 50%
    +rwdw-max()
        width: 100%
    CSS 
    .test {
    width: 50%;
}
@media all and (max-width: 1024px) {
    .test {
        width: 100%;
    }
}
  2. @media all and (min-width) 
    +rwdw-min(val)

    使用方式：

    SASS 
    .test
    width: 50%
    +rwdw-min()
        width: 100%
    CSS 
    .test {
    width: 50%;
}
@media all and (min-width: 1025px) {
    .test {
        width: 100%;
    }
}
  3. @media all and (min-width) and (max-width) 
    +rwdw-range(max,min)

    使用方式： 

    +rwdw-range(1024,768)

上頭提到的$rwd 等變數可在_config.sass修改設定。

預設值： 

$rwd: 1024 //Menu轉換手機尺寸
$rwd-pc: 1280 //電腦尺寸
$rwd-pcs: 1024 //小電腦尺寸
$rwd-pad: 992 //平板尺寸
$rwd-pads: 768 //小平板尺寸
$rwd-phone: 480 //手機尺寸
$rwd-phones: 320 //小手機尺寸
