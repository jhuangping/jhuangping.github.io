rem()

px to rem

預設值 _config.sass $default: htmlFz: 10 ，文字預設 font-size: 10px。 使用方式： rem(val, Word level)

以下範例：

SASS 
.test
    font-size: rem(16)
CSS 
.test {
    font-size: 1.6rem;
}
em()

px to em

預設值 _config.sass $default: bodyFz: 16 ，文字預設 font-size: 16px。使用方式： em(val, Parent level)

以下範例：當 .test 父層字級為 16px 時

SASS 
.test
    font-size: em(16)
CSS 
.test {
    font-size: 1em;
}
vw()

px to vw

預設最大寬度 1920, 使用方式: vw(val, Screen Width)

SASS 
.test
    width: vw(1920)
.test2
    width: vw(860)

// 重設尺寸
.test
    width: vw(512, 1024)
CSS 
.test { width: 100vw; }
.test2 { width: 50vw; }

// 重設尺寸
.test { width: 50vw; }

media

段點 @media

針對指定的「媒體」設定樣式，這裡使用的媒體類型為" all " ( 用於所有設備 )。

裡頭的$rwd等變數可在 _config.sass 修改設定。

預設值：

變數 $rwd $rwd-pc $rwd-pcs $rwd-pad $rwd-pads $rwd-phone $rwd-phones
預設值 1024 1366 1200 992 768 575 320

在這裡有三個可用分別是

+rwdw-max()
@media all and (max-width) 
+rwdw-max(val)

使用方式：

SASS 
.test
    width: 50%
    +rwdw-max()
        width: 100%
CSS 
.test {
    width: 50%;
}
@media all and (max-width: 1024px) {
    .test {
        width: 100%;
    }
}
+rwdw-min()
@media all and (min-width) 
+rwdw-min(val)

使用方式：

SASS 
.test
    width: 50%
    +rwdw-min()
        width: 100%
CSS 
.test {
    width: 50%;
}
@media all and (min-width: 1025px) {
    .test {
        width: 100%;
    }
}
+rwdw-range()
@media all and (min-width) and (max-width) 
+rwdw-range(max,min)

使用方式：

SASS 
// Default
+rwdw-range() 
// Custom
+rwdw-range(1024,768)
CSS 
// Default
@media all and (min-width: { setting value }px) and (max-width: 1920px) ...
// Cutsom
@media all and (min-width: 769px) and (max-width: 1024px) ...
