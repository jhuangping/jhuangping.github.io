前端套件管理，Bower 的運用

Bower 環境(Windows 安裝 Node.js)

  • Twitter 團隊開發的前端開發套件管理工具
  • 管理/安裝 前端開發所需要的package (css/javascript)
  • 自動依套件相依性安裝
  • 以往開發很常遇到的情境：

    專案使用了許多外部工具(例如 jquery, bootstrap ...)
    每隔一陣子想更新到最新版，就要分別下載更新
    並且可能會有相容性問題要處理

  • bower 可以解決以上問題

Bower 相關指令

  • bower -v // 查詢當前bower的版本
  • npm install bower -g // 全域安裝 bower
  • bower install // 查專案安裝套件
  • bower list // 查詢專案目前安裝的套件
  • bower update // 更新套件
  • bower search // 搜尋套件

Bower 安裝步驟

  1. 安裝 Node.js

    安裝 node.js 也會順便把 npm 安裝進去，接著就是使用 windows 的 cmd 或 Mac 的 Termial 來安裝 Bower，首先我們先輸入以下的指令，把 Bower安裝到全域的環境。( 在 Mac 要記得加 sudo 不然會裝不進去 )

    依作業系統選擇Node.js版本

  2. 首先要透過npm 全域安裝 bower
    npm install bower -g
  3. 創建bower設定檔 // 第一次建立
    bower init
